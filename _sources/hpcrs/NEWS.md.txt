# Release notes for HPCRS

## Release `v0.2.0` :

Initial release of the High Performance Compute Reference Stack (HPCRS). This
release features the `sysstacks/hpcrs-centos` Docker image available on
DockerHub*.

- Components included:
  - CentOS
  - PyTorch*
  - Horovod*
  - OpenMP
  - Intel® MPI Library
  - Intel® C++ Compiler
  - Intel® Fortran Compiler 
  - Intel® Math Kernel Library
  - Intel® Data Parallel C++
  - Intel® oneAPI Collective Communications Library
  - Intel® oneAPI Threading BUilding Blocks
  - libfabric
  - Spack

- Versions:
  - CentOS -> `8`
  - PyTorch ->  `1.7.0`
  - Horovod ->  `0.19.5`
  - OpenMP -> `4.5`
  - Intel® MPI Library ->  `2019.7.217`
  - Intel® C++ Compiler ->  `parallel_studio_xe_2020_update1_composer_edition`
  - Intel® Fortran Compiler ->  `parallel_studio_xe_2020_update1_composer_edition`
  - Intel® Math Kernel Library ->  `2020.1.217`
  - Intel® oneAPI Data Parallel C++ Compiler -> `2021.1`
  - Intel® oneAPI Collective Communications Library -> `2021.1.1`
  - Intel® oneAPI Threading BUilding Blocks -> `2021.1.1`
  - libfabric -> `v1.11.1`
  - Spack -> `0.15.4`
